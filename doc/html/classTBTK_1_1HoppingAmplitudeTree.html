Detailed Description

HoppingAmplitudeTree is a tree structure used to build a tree for stroing HoppingAmplitudes . Used by AmplitudeSet.

TBTK::HoppingAmplitudeTree::HoppingAmplitudeTree ( const std::vector< unsigned int > &  capacity)

Constructor. Preallocates a HoppingAmplitudeTree such that the addition of HoppingAMplitudes with indices that have the same subindex structure as 'capacity', but with smaller subindices will not cause reallocation for the main tree stucture. Internal containers for HoppingAMplitudes may still be reallocated.
